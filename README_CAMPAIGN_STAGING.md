# 🎭 Campaign Staging Area

Staging area for campaign changes where you can review and approve them before they go live. Integrates with guardrails for safety validation.

## 🚀 **What It Does:**

### **✅ Change Management:**
1. **Propose Changes** - Budget, targeting, and asset modifications
2. **Guardrails Validation** - Automatic safety checks before staging
3. **Review & Approve** - Review all changes before implementation
4. **Staged Execution** - Changes execute only after approval
5. **Change History** - Track all approved, rejected, and pending changes

### **✅ Safety Integration:**
1. **Budget Guardrails** - Min/max limits, adjustment percentages, frequency controls
2. **Targeting Guardrails** - Location restrictions, demographic limits
3. **Asset Guardrails** - PMax requirements, format validation
4. **Execution Timing** - 2-hour change window for safety
5. **One Lever Per Week** - Prevents over-optimization

## 🛠️ **Features:**

### **📋 Pending Changes Tab:**
- **Review All Changes** - See all pending modifications
- **Guardrails Status** - View validation results
- **Approve/Reject** - Make decisions on each change
- **Modify Changes** - Edit proposals before approval

### **💰 Budget Changes Tab:**
- **Current Budget Display** - See existing campaign budgets
- **New Budget Proposal** - Propose budget adjustments
- **Reason Selection** - Document why changes are needed
- **Effective Date** - Schedule when changes take effect
- **Guardrails Validation** - Automatic safety checks

### **🎯 Targeting Changes Tab:**
- **Location Management** - Add/remove geographic targeting
- **Demographic Adjustments** - Age, income, interest targeting
- **Negative Keywords** - Exclude irrelevant traffic
- **Audience Optimization** - Refine target audiences

### **🖼️ Asset Changes Tab:**
- **Asset Type Selection** - Images, videos, text, logos
- **Action Management** - Add, remove, or update assets
- **Asset Source** - Website scraping, upload, AI generation
- **Quantity Control** - Specify number of assets to change

## 📋 **Workflow:**

### **1. Propose Changes:**
- Navigate to appropriate tab (Budget, Targeting, Assets)
- Fill out change details
- Click "Validate with Guardrails"

### **2. Review Validation:**
- Check guardrails approval status
- Review any warnings or rejections
- Modify proposal if needed

### **3. Approve Changes:**
- Go to "Pending Changes" tab
- Review all proposed changes
- Approve or reject each change
- Changes execute after approval

### **4. Monitor Execution:**
- Track approved changes
- Monitor performance impact
- Adjust based on results
